# Antigravity Demo - Quick Start

This workflow gets you up and running with the Antigravity Demo application.

## Prerequisites

- Node.js 18+ installed
- npm package manager

## Steps

1. Navigate to the project directory

```bash
cd c:\Users\scarm\.gemini\antigravity\scratch\antigravity-demo
```

2. Install dependencies (if not already installed)

// turbo
```bash
npm install
```

3. Start the development server

// turbo
```bash
npx vite
```

4. Open your browser to the displayed local URL (typically http://localhost:5173 or http://localhost:5174)

## What You'll See

- **Particle Background**: Animated particles that respond to mouse movement
- **Theme Toggle**: Switch between dark and light modes
- **Interactive Counter**: Test React state management
- **Features Showcase**: Antigravity capabilities
- **MCP Integration**: Display of active MCP servers
- **Firebase Card**: Firebase capabilities showcase
- **Workflows Card**: Available automation workflows
